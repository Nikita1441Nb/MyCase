Проект: Rentify

Комплексная CRM-система для прокатного бизнеса

О проекте

Rentify — это кастомная CRM, разработанная с нуля для действующего предприятия. Весь цикл создания продукта — от проектирования UI/UX интерфейса до настройки серверов — реализован мной. Система создавалась в формате 24/7 контакта с заказчиком: требования собирались и внедрялись прямо на ходу, чтобы решить реальные боли менеджеров.

Главным фокусом была максимальная скорость обслуживания и независимость от качества интернета, поэтому ядро работает с локальной базой данных без ограничений по количеству оборудования.

Технологический стек

Python / Django REST API Node.js / Electron HTML / CSS / JS UI/UX PostgreSQL Docker Gunicorn + Nginx

Ключевые решения и бизнес-результат

  • Мгновенное обслуживание (UI/UX + Логика): Оформление одного клиента теперь занимает всего 30 секунд. Этого удалось достичь благодаря внедрению функции сканирования данных по фото и интерфейсу, который я спроектировал специально под нужды менеджеров.
  • Автономный десктоп-клиент: Упаковка веб-интерфейса в нативное приложение через Electron и Node.js. Локальная база данных гарантирует моментальный отклик и работу без привязки к стабильности интернета.
  • Интеграция с Telegram: Разработан бот, который берет данные из ядра CRM и регулярно отправляет выгрузки и статистику владельцу бизнеса для контроля.
  • DevOps и Инфраструктура: Проект полностью упакован в Docker. Самостоятельно поднял и настроил VPS, развернул связку Gunicorn + Nginx, подключил домен и обеспечил безопасность веб-приложения через Cloudflare.
